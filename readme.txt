== Description ==

This BuddyPress RC2 plugin simply removes the default Admin Bar. It's ready-to-use: just activate it; no options needed.

If you find this plugin useful, please rate it.

== Installation ==

1. Upload `arit_remove_buddypress_admin_bar.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
